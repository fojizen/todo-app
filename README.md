# 🌌 Personal Task List with Neon Aesthetic

A sleek, interactive To-Do List application featuring a modern "Moody" design, particle backgrounds, and a secure user authentication simulation. Built to help users stay organized while enjoying a high-performance visual experience.

## ✨ Features
* **User Auth System:** Simple Login and Register interface.
* **Persistent Storage:** Tasks are saved to `localStorage`, so they won't disappear when you refresh the page.
* **Visual Aesthetics:** Neon glow effects, smooth transitions, and dynamic particle background.
* **Responsive Design:** Optimized for both desktop and mobile devices.
* **Interactive UI:** Custom cursor effects and "Glitch" animations.

## 🛠️ Built With
* **HTML5** - Structural foundation.
* **CSS3** - Custom styling with CSS Variables and Animations.
* **JavaScript (ES6+)** - Logic, DOM manipulation, and storage management.
